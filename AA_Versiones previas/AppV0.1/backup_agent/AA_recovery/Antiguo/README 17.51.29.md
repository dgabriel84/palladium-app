# Palladium Reservas - Sistema de Prediccion

Aplicacion web para reservas de hotel con prediccion de cancelaciones.

## Instalacion

```bash
pip install streamlit pandas
```

## Ejecucion

```bash
cd 03_app
streamlit run app.py
```
